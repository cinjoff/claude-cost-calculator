# Claude vs Amsterdam

> Is Claude expensive? Let's put it in Amsterdam terms.

A one-pager that reframes the Claude Pro/Max subscription cost as everyday Amsterdam spending — beers, coffees, cinema tickets, and a few other local Amsterdam staples.

## Running locally

```bash
npm install
npm run dev
```

## Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Framer Motion
- TypeScript
- Deployed on Vercel
